Help

This simple monitoring tool calculates the estimated income [excl. hosting] of your Dash Masternodes and displays it in a clear way. The data gets refreshed every minute automatically. To learn more about the Dash project visit Dash.org.

Url parameters

Default parameters:
1000 Dash (1MN) and USD as fiat currency.

Custom parameters:
curr=EUR or CNY,GBP,CAD,RUB,HKD,JPY,AUD,CHF,PLN
addr=address1,address2,address3... (too many addresses will take some time)
OR
value=customDashAmount
